### 🚀 Флагманские проекты

#### 1. ProcurePro — Система автоматизации закупок
**Стек:** Django 4.2, Django REST Framework, Celery, Redis, JWT, PostgreSQL

Полноценная backend-система для автоматизации закупок в розничной сети. То, что я делал руками как управленец (договаривался с поставщиками, считал закупки, контролировал остатки), теперь можно автоматизировать.

**Что реализовано:**
- Многоранговая система пользователей (клиенты, поставщики, администраторы)
- Каталог товаров с иерархией категорий и характеристиками
- Корзина с поддержкой товаров от разных поставщиков в одном заказе
- Полноценный цикл заказов со статусами и историей
- Асинхронные задачи через Celery (email-уведомления, импорт/экспорт)
- REST API с полной документацией (Swagger)
- Админ-панель для управления всеми сущностями

#### 2. Voice Synthesizer — Профессиональный голосовой синтезатор
**Стек:** Python 3.12, PyQt5, PyTorch, Silero TTS, sounddevice, NLTK, PyInstaller

Профессиональное desktop-приложение для преобразования текста в речь с использованием нейросетевых технологий. Полностью автономная программа с интуитивным интерфейсом.

**Ключевые возможности:**
- 🎙️ **6 голосов на выбор** (aidar, baya, kseniya, xenia, eugene, random)
- 📝 **Редактор текста** с форматированием (шрифты, размеры, цвета, жирный)
- 🔤 **T9 предсказание слов** с обучением на основе ввода пользователя
- 🎚️ **Настройка скорости** произношения (50-200%)
- 💾 **Сохранение** синтезированной речи в WAV формат
- 🔍 **Масштабирование интерфейса** (50-400%)
- 💻 **Полная автономность** — работает без интернета

**Техническая реализация:**
- Нейросетевой синтез речи на базе Silero TTS
- Конкатенативный, параметрический и нейросетевой методы синтеза
- Пост-обработка аудио для улучшения качества окончаний
- Автоматическое сохранение настроек между сеансами
- Готовый установщик (Inno Setup) и портативная версия

**Почему этот проект важен:** Я реализовал сложное desktop-приложение с нуля, используя современные технологии машинного обучения. Программа полностью соответствует техническому заданию и готова к промышленному использованию.

👉 [Ссылка на репозиторий Voice Synthesizer](https://github.com/nmak1/voice-synthesizer)
